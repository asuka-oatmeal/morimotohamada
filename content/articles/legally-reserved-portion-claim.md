# 遺留分を請求したい！計算から時効、手続きまで専門家が徹底解説

「遺言書の内容に納得できない」「自分だけ財産がもらえないのはおかしい」と感じていませんか？

もし、亡くなった親や配偶者（被相続人）の遺言によって、あなたが本来受け取るべき相続財産の一部をもらえなかった場合、「遺留分」を請求できる可能性があります。遺留分は、特定の相続人に保障された最低限の遺産の取り分であり、法的に守られた強い権利です。

しかし、「遺留分って何？」「どうやって請求するの？」「いつまでに請求すればいいの？」といった疑問をお持ちの方も多いでしょう。遺留分請求は、正しい知識と手順を踏まないと、大切な権利を失ってしまうことにもなりかねません。

この記事では、日本の法律に詳しいSEOライターが、遺留分の基本的な知識から、具体的な[計算方法](/blog/child-support-calculation)、請求手続きの流れ、時効、さらにはよくあるトラブルと解決策まで、遺留分請求に関するあらゆる情報をわかりやすく解説します。専門用語は避け、具体的な数字や事例を交えながら、一般の方が安心して遺留分請求を進められるようサポートします。

あなたの正当な権利を守り、納得のいく相続を実現するために、ぜひこの記事を最後までお読みください。

## 遺留分とは？基礎知識を徹底解説

まずは、遺留分がどのような権利なのか、その基本的な概念と、誰が請求できるのかについて確認していきましょう。

### 遺留分の概念と民法改正

遺留分とは、被相続人の兄弟姉妹以外の法定相続人に保障されている、遺産の最低限の取り分のことです。被相続人は、遺言によって自分の財産を自由に処分できますが、その自由は無制限ではありません。一部の相続人（遺留分権利者）には、生活保障や相続人間での公平性を図るため、最低限の財産を取得する権利が与えられています。

2019年7月1日に施行された改正民法により、遺留分の制度は大きく変わりました。それまでの「遺留分減殺請求権」は「遺留分侵害額請求権」へと名称が変更され、遺留分を侵害された相続人は、侵害額に相当する金銭の支払いを請求できるようになりました。改正前は、遺留分に相当する現物財産（不動産など）を直接取り戻すことが可能でしたが、現物を共有する形となり、かえってトラブルの原因となることも少なくありませんでした。

改正により金銭請求となったことで、以下のようなメリットがあります。

*   **解決の迅速化:** 現物分割に伴う複雑な評価や共有関係の解消が不要になり、金銭での解決が容易になりました。
*   **相続人間のトラブル軽減:** 不動産などの共有を避けることで、その後の関係悪化を防ぎやすくなりました。

### 遺留分を請求できる人は誰？（遺留分権利者）

遺留分を請求できる人（遺留分権利者）は、以下の法定相続人に限られます。

| 関係性           | 遺留分権利者となるか | 備考                               |
| :--------------- | :------------------- | :--------------------------------- |
| **配偶者**       | 〇                   | 常に遺留分権利者となる             |
| **子（直系卑属）** | 〇                   | 常に遺留分権利者となる             |
| **親（直系尊属）** | 〇                   | 子がいない場合に遺留分権利者となる |
| **兄弟姉妹**     | ×                    | 遺留分権利者ではない               |

**ポイント:**
*   兄弟姉妹には遺留分が認められていません。
*   子が複数いる場合は、それぞれの子が遺留分権利者となります。
*   子がすでに亡くなっている場合は、孫（代襲相続人）が遺留分権利者となります。
*   直系尊属（親、祖父母）は、子や孫などの直系卑属がいない場合にのみ遺留分権利者となります。

### 遺留分が認められないケースもある？

遺留分は強力な権利ですが、例外的に認められないケースもあります。

1.  **遺留分放棄:** 遺留分権利者が、被相続人の生前に家庭裁判所の許可を得て遺留分を放棄した場合、または相続開始後に遺留分侵害額請求権を放棄した場合。
2.  **相続欠格・相続人廃除:** 遺留分権利者が、被相続人に対して重大な犯罪行為を行ったなど、民法上の「相続欠格事由」に該当する場合や、被相続人が家庭裁判所に申し立てて「相続人廃除」が認められた場合。
3.  **時効の成立:** 後述しますが、遺留分侵害額請求権には厳格な時効期間が定められています。この期間を過ぎてしまうと、遺留分を請求する権利は消滅してしまいます。

## あなたの遺留分はいくら？具体的な計算方法と事例

次に、実際にあなたの遺留分がいくらになるのか、具体的な計算方法を見ていきましょう。遺留分の計算は、まず「全体の遺留分割合」を算出し、次に「遺留分侵害額の算定基礎財産」を確定させ、最後に「個別の遺留分額」を算出するという手順で進めます。

### 遺留分の割合は法定相続分とどう違う？

まず、全体の遺留分割合は、誰が相続人になるかによって決まります。これは、民法で定められている「法定相続分」とは異なるので注意が必要です。

| 遺留分権利者               | 全体の遺留分割合 | 各遺留分権利者の遺留分割合 |
| :------------------------- | :--------------- | :------------------------- |
| 配偶者のみ                 | 2分の1           | 配偶者：2分の1             |
| 子のみ                     | 2分の1           | 子：それぞれが2分の1を等分 |
| 配偶者と子                 | 2分の1           | 配偶者：4分の1、子：4分の1を等分 |
| 親（直系尊属）のみ         | 3分の1           | 親：それぞれが3分の1を等分 |
| 配偶者と親（直系尊属）     | 2分の1           | 配偶者：3分の1、親：6分の1を等分 |

**補足:** 上記の「各遺留分権利者の遺留分割合」は、全体の遺留分割合を、その権利者構成における法定相続分で按分した割合になります。

### 遺留分侵害額の算定基礎財産とは？

遺留分の計算の元となる財産を「算定基礎財産」といいます。これは、被相続人が亡くなった時点（相続開始時）に持っていた財産に、特定の贈与を加え、そこから負債を差し引いて算出されます。

**算定基礎財産 = (相続開始時の財産) + (特定の贈与) - (相続債務)**

それぞれの要素について詳しく見ていきましょう。

1.  **相続開始時の財産:**
    *   預貯金、不動産、有価証券、自動車、貴金属などのプラスの財産全てです。
    *   生命保険金や死亡退職金は、原則として遺留分算定基礎財産に含まれませんが、特定の受取人が取得した保険金が極めて高額で、相続財産全体の金額とのバランスを著しく欠く場合は、例外的に含まれることがあります（特別受益に準ずるものとして扱われる）。

2.  **特定の贈与:**
    *   **相続人への贈与:** 相続開始前10年以内に行われた贈与（特別受益に該当するもの）。
    *   **相続人以外への贈与（第三者への贈与）:** 相続開始前1年以内に行われた贈与。ただし、遺留分権利者に損害を加えることを知って行った贈与は、期間の制限なく算入されます。

3.  **相続債務（負債）:**
    *   借金、未払金、連帯保証債務など、被相続人が残したマイナスの財産です。葬儀費用は、原則として相続債務には含まれません。

### 計算シミュレーション例

具体的な事例で計算方法を確認しましょう。

**【事例設定】**
被相続人：父（他界）
相続人：母（配偶者）、長男、次男
遺産総額：5,000万円（自宅不動産3,000万円、預貯金2,000万円）
父の債務：なし
父の遺言：全財産を長男に相続させる。

この場合、遺留分を侵害されているのは母と次男です。

1.  **算定基礎財産:** 5,000万円（特定の贈与・債務がないため）
2.  **全体の遺留分割合:** 相続人が配偶者と子なので「2分の1」
    *   全体の遺留分額 = 5,000万円 × 1/2 = 2,500万円
3.  **各遺留分権利者の割合:**
    *   母（配偶者）の法定相続分は2分の1、子の法定相続分はそれぞれ4分の1。
    *   全体の遺留分2,500万円をこの割合で按分します。
        *   母の遺留分侵害額 = 2,500万円 × 1/2（配偶者の法定相続分）= 1,250万円
        *   長男・次男それぞれの遺留分侵害額 = 2,500万円 × 1/4（子の法定相続分）= 625万円

**結論:**
*   母は長男に対し、遺留分として1,250万円を請求できます。
*   次男は長男に対し、遺留分として625万円を請求できます。

長男は遺言により5,000万円を相続しましたが、母と次男から合計1,875万円の遺留分侵害額請求を受けることになります。

### 特別受益や寄与分はどう影響する？

*   **特別受益:** 相続人のうち特定の人が、被相続人から生前贈与や遺贈によって特別に受けた利益のことです。遺留分侵害額の計算においては、上記の算定基礎財産に算入される「特定の贈与」として考慮されます。
    *   例: 親から住宅購入資金として贈与を受けていた場合、その金額は算定基礎財産に加えられ、さらにその贈与を受けた相続人の遺留分額から控除されます。これにより、相続人間の公平性が図られます。

*   **寄与分:** 特定の相続人が、被相続人の財産の維持・増加に特別に貢献した場合に認められるものです。遺留分算定基礎財産を計算する際には、寄与分に相当する額が基礎財産から控除されることになります。
    *   例: 長年、被相続人の介護を無償で行い、財産の減少を防いだ場合など。

これらの考慮により、遺留分侵害額はより複雑になるため、判断に迷う場合は専門家への相談が不可欠です。

## 遺留分を請求するには？手続きの流れと注意点

いざ遺留分を請求しようと思っても、具体的にどうすれば良いのか迷う方もいるでしょう。ここでは、請求手続きの流れと、特に注意すべき点を解説します。

### 遺留分侵害額請求権の行使とは？（意思表示の方法）

遺留分侵害額請求権を行使するために、特別な手続きや裁判は必要ありません。法的には、遺留分権利者が、遺留分を侵害している相手方（受遺者や受贈者）に対して、「遺留分を請求する」という意思表示をするだけで効力が発生します。

しかし、口頭での意思表示は、後々「言った」「言わない」のトラブルになりかねません。そのため、**内容証明郵便**を利用して意思表示を行うことが強く推奨されます。

**内容証明郵便のメリット:**
*   **相手に意思表示した証拠が残る:** いつ、どのような内容の意思表示をしたのかを郵便局が証明してくれます。
*   **時効の中断効果:** 内容証明郵便を送ることで、後述する時効期間を一時的に中断させることができます。

内容証明郵便には、以下の内容を記載します。
*   請求の趣旨（遺留分侵害額を請求する旨）
*   算定基礎財産の内訳と遺留分額
*   支払いを求める具体的な金額と期限

### 請求期限（時効）は厳守！

遺留分侵害額請求権には、厳格な請求期限（時効）が定められています。この期間を過ぎてしまうと、どれだけ正当な権利があっても請求できなくなってしまいます。

| 期限の種類       | [いつから](/blog/joint-custody-2026-overview)カウントされるか                                              | 期間    |
| :--------------- | :-------------------------------------------------------------------- | :------ |
| **短期消滅時効** | 遺留分権利者が、相続の開始と遺留分を侵害する贈与または遺贈があったことを知った時から | **1年** |
| **長期消滅時効** | 相続開始の時から（遺留分権利者が上記を知らなくても）                  | **10年** |

**最も重要なのは「1年」の短期消滅時効です。**
被相続人が亡くなり、遺言書の内容を知って自分の遺留分が侵害されていることに気づいたら、速やかに対応しなければなりません。

**注意点:**
*   遺言書の内容を、弁護士や他の相続人から伝えられた場合でも、「知った時」と見なされることがあります。
*   時効期間が迫っている場合は、内容証明郵便の発送だけでも早急に行いましょう。内容証明郵便を送ることで、時効の完成が6ヶ月間猶予される効果があります（時効の完成猶予）。この期間内に、調停や訴訟を起こす必要があります。

### 遺留分を請求する相手は誰？

遺留分を請求する相手は、**遺留分を侵害している「受遺者」または「受贈者」**です。

*   **受遺者:** 遺言によって財産を受け取った人。
*   **受贈者:** 生前贈与によって財産を受け取った人。

例えば、遺言で全財産を長男に相続させると書かれていた場合、遺留分を侵害された他の相続人（配偶者や次男など）は、長男に対して遺留分を請求することになります。

### 請求後の交渉から調停・訴訟へ

意思表示後、相手方が速やかに支払いをしてくれれば良いのですが、多くの場合、交渉が必要になります。

1.  **話し合い・交渉:**
    *   まずは、内容証明郵便の送付後、相手方との間で直接話し合いを行います。
    *   遺留分額の計算、支払い方法、支払い時期などについて合意形成を目指します。
    *   合意できた場合は、後々のトラブル防止のため、合意内容を記した「合意書」を作成し、公正証書にしておくことも検討しましょう。

2.  **遺留分侵害額請求調停:**
    *   話し合いで解決できない場合は、家庭裁判所に「遺留分侵害額請求調停」を申し立てます。
    *   調停では、調停委員が間に入り、双方の主張を聞きながら合意形成を促します。
    *   弁護士を代理人として立てることで、専門的な交渉を進められます。

3.  **遺留分侵害額請求訴訟:**
    *   調停でも合意に至らなかった場合、最終手段として「遺留分侵害額請求訴訟」を提起することになります。
    *   訴訟では、裁判官が双方の提出する証拠に基づいて判断を下し、判決を言い渡します。
    *   訴訟は専門的な知識が求められ、時間や費用もかかるため、弁護士に依頼することが一般的です。

### 弁護士に依頼するメリット・デメリット

遺留分請求は、法律知識が不可欠な手続きであり、感情的な対立も生じやすいため、弁護士への依頼を検討する方も多いでしょう。

**弁護士に依頼するメリット:**

*   **複雑な計算や法的手続きを代行:** 算定基礎財産の計算、内容証明郵便の作成・送付、調停・訴訟手続きなど、全て任せられます。
*   **相手方との交渉代理:** 感情的な対立を避け、冷静かつ有利な交渉を進めることができます。
*   **時効管理の徹底:** 厳格な時効を失念するリスクを回避できます。
*   **精神的負担の軽減:** 専門家に任せることで、精神的なストレスを大きく減らせます。
*   **最適な解決策の提示:** 個別の事情に応じた最も有利な解決策を提案してもらえます。

**弁護士に依頼するデメリット:**

*   **費用がかかる:** 着手金、報酬金、実費などの弁護士費用が発生します。
*   **弁護士選びに時間:** 信頼できる弁護士を見つける必要があります。

費用の問題はありますが、遺留分請求の成功率や得られる金額を考慮すると、弁護士に依頼するメリットは大きいと言えるでしょう。

## 遺留分請求でよくあるトラブルと解決策

遺留分請求は、多くの場合、親族間での争いになるため、感情的な対立が生じやすく、様々なトラブルが発生する可能性があります。ここでは、よくあるトラブルとその解決策を紹介します。

### 1. 相手が請求に応じない場合

相手方が遺留分侵害を認めなかったり、支払いを拒否したりするケースは少なくありません。

*   **解決策:** まずは内容証明郵便で明確な意思表示を行い、それでも応じない場合は、家庭裁判所に**遺留分侵害額請求調停**を申し立てます。調停でも解決しない場合は、**訴訟**へと移行することになります。この段階では、弁護士のサポートが不可欠です。

### 2. 財産の評価で揉める場合

不動産や未公開株式など、客観的な評価が難しい財産がある場合、その評価額を巡って争いになることがあります。遺留分侵害額の算定基礎となる財産の評価額が異なれば、最終的な請求額も大きく変わるため、重要な争点となります。

*   **解決策:** 専門家（不動産鑑定士、公認会計士など）による**鑑定評価**を活用します。調停や訴訟では、裁判所が選任した鑑定人による評価がなされることもあります。弁護士を通じて、適切な評価額を主張していくことになります。

### 3. 特別受益や寄与分の主張が対立する場合

「生前にもらったお金は特別受益ではない」「長年の介護は寄与分として評価されるべき」など、他の相続人が特別受益や寄与分を主張し、遺留分侵害額の計算に影響を与えるケースがあります。

*   **解決策:** 特別受益や寄与分が認められるかどうかは、法的な要件に基づき判断されます。贈与の目的、金額、被相続人の財産状況、介護の内容などを詳細に立証する必要があります。弁護士は、これらの主張が正当であるか否かを判断し、適切な反論や立証を行います。

### 4. 遺言書自体に問題がある場合

遺言書が偽造されたものではないか、被相続人が認知症などで遺言能力がなかったのではないか、などの疑義が生じることもあります。

*   **解決策:** 遺言書の有効性を争う「遺言無効確認訴訟」を提起することも可能です。ただし、遺言書の無効を主張するには、客観的な証拠が必要であり、非常に難易度の高い訴訟となります。まずは弁護士に相談し、遺言書の有効性を争うべきか、あるいは遺留分侵害額請求に注力すべきかを検討しましょう。

## 遺留分請求にかかる費用と税金

遺留分を請求する際に気になるのが、手続きにかかる費用と、請求によって得た金銭にかかる税金でしょう。

### 弁護士費用相場

弁護士費用は、事務所や案件の複雑さによって異なりますが、一般的な相場は以下のようになります。

| 費用の種類 | 内容                                                         | 相場（一般的な目安）                                     |
| :--------- | :----------------------------------------------------------- | :------------------------------------------------------- |
| **法律相談料** | 初回の相談にかかる費用                                       | 0円～1万円/30分～1時間（無料相談を設けている事務所も多い） |
| **着手金** | 弁護士が案件に着手する際に支払う費用（結果に関わらず返還されない） | 遺留分請求額に応じて変動（例: 20万円～50万円程度）        |
| **報酬金** | 遺留分請求が成功した場合に、経済的利益に応じて支払う費用     | 経済的利益の10%～20%程度（例: 獲得額の16%＋消費税）      |
| **実費**   | 郵便代、交通費、印紙代、鑑定費用など、手続きにかかる費用     | 数万円～数十万円                                         |

多くの弁護士事務所では、経済的利益（最終的に相手から得られた金額）に応じて着手金や報酬金を算定する料金体系を採用しています。

### 調停・訴訟費用

弁護士費用とは別に、裁判所に支払う費用もあります。

*   **調停申立費用:** 数千円～1万円程度（遺留分請求額による）
*   **訴訟提起費用（印紙代）:** 請求額に応じて変動（例: 100万円請求なら1万円、500万円請求なら3万円など）
*   **郵券代:** 数千円

これらの費用は、調停や訴訟を進める上で必ず発生するものです。

### 遺留分請求で得た財産に税金はかかる？

遺留分侵害額請求によって受け取った金銭は、**相続税の課税対象**となる可能性があります。

これは、遺留分として受け取った金銭が、本来被相続人から相続するはずだった財産の一部と見なされるためです。したがって、相続税の基礎控除額を超える金額を受け取った場合は、相続税が課税されることになります。

**ポイント:**
*   遺留分を取得した相続人は、他の相続財産と合算して相続税を計算し、申告・納税する義務があります。
*   税金の計算や申告は複雑なため、相続税専門の税理士に相談することをお勧めします。

## まとめ

遺留分請求は、亡くなった方が残した遺言によって、自身の正当な相続分を侵害された場合に、最低限の財産を取り戻すための重要な権利です。

この記事では、遺留分請求について以下のポイントを解説しました。

*   **遺留分とは:** 特定の相続人に保障された最低限の取り分で、改正民法により金銭請求が可能に。兄弟姉妹以外の法定相続人が請求できます。
*   **請求できる人:** 配偶者、子（代襲相続人含む）、直系尊属（親、祖父母など）が遺留分権利者です。
*   **計算方法:** 相続開始時の財産に特定の贈与を加え、債務を引いた「算定基礎財産」に、各相続人の「遺留分割合」を掛けて算出します。具体的な事例で計算例を提示しました。
*   **請求手続き:**
    *   まずは「**内容証明郵便**」で意思表示を行うことが重要。
    *   **時効**は、相続開始と遺留分侵害を知ってから**1年**、または相続開始から**10年**。この期間を厳守する必要があります。
    *   相手方との交渉で解決しない場合、家庭裁判所の「**調停**」、それでも解決しない場合は「**訴訟**」へと進みます。
*   **弁護士に依頼するメリット:** 複雑な手続きの代行、交渉代理、時効管理、精神的負担の軽減など、多岐にわたります。
*   **よくあるトラブル:** 相手が請求に応じない、財産評価で揉める、特別受益・寄与分で対立するなどが挙げられます。
*   **費用と税金:** 弁護士費用や裁判所費用がかかり、請求で得た金銭には相続税が課税される場合があります。

遺留分請求は、法的な知識が求められ、時には感情的な対立も伴う複雑な手続きです。一人で抱え込まず、法律の専門家である弁護士に相談することが、スムーズかつ有利な解決への近道となります。

「遺留分 請求」について疑問や不安がある方は、まずは無料相談などを活用し、専門家のアドバイスを仰ぐことを強くお勧めします。あなたの正当な権利を守り、納得のいく相続を実現するための一歩を踏み出しましょう。
